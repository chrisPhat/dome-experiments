THREE.js Dome Experiments

This web page is dedicated to my experiments in building content for DomeLab.

The Demos:

Renderer

The Renderers:

  • WebVR - Preview scene in virtual reality with depth.
  • WebVRDome - Preview "flat" scene in a virtual reality dome theater.
  • Panorama - Equirectangular 360 panorama for projection.
  • FullDome - Hemispherical image for projection.

Supported VR Headsets:

The two WebVR renderers should work on all major headsets, including Cardboard.

GearVR: You must download the Samsung Internet Browser for Gear VR from the Oculus Store Android app. You must then visit this webpage while in VR mode.

Oculus Rift and HTC Vive: Rift and Vive users must visit this page using an experimental WebVR version of Chrome or Firefox.

Resources:

  • THREE.CubemapToEquirectangular - Code from which I built the PanoramaRenderer.
  • CefWithSyphon - Special build of Chrome that can send content to Blendy Dome VJs via Syphon
  • Blendy Dome VJs - Dome projection software that can project Syphon content to dome.
  • WebVR - Information on WebVR used in the preview renderers
  • THREE.js - The JavaScript graphics library used in these demos
